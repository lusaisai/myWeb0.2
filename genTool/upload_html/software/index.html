Linux及其软件推荐

下面列了一些Linux上较好的软件。

CentOS 和 Linux Mint

先说操作系统吧。

如果说要推荐个人操作系统的话，我会推荐Linux Mint, linuxmint是基于ubuntu开发的，ubuntu本身是个很好的系统，有canonical在后面赞助着，只是ubuntu从2011年开始使用unity用户界面，也许再过几年，unity会变得很好，但目前看，问题很多，许多老用户也很不习惯，所以并不很推荐。相反，linuxmint的开发一直在听取用户的意见，有各种桌面环境可以选择，像Cinnamon和MATE等等，强烈推荐！

说到企业级系统，那就要说CentOS了,可以拿来当server，也可以当桌面系统。CentOS基本是Redhat的克隆，不过是免费的，我自己电脑上装的就是CentOS, 响应很快，非常稳定，我很喜欢。

ThunderBird

邮件客户端，很强大，可以安装插件实现各种功能。我很喜欢它的图标，你一看就懂了。

GIMP

Linux上的photoshop，最近刚刚release了一个新版本gimp2.8，如果觉得photoshop太贵的话，可以考虑gimp，该有的功能都有。

Audacious

轻巧的音乐播放器，略像foobar2000，小而强大。

VLC 和 SMPlayer

两款视频播放器，都很强。看形势，将来估计就是VLC的天下了，因为开发的进度非常快。SMPlayer是mplayer的图形界面，因为mplayer的命令行控制会让一些人不适应，有个图形前端，用户体验好点。

GNU Utils 和 Pakages

然后就是各种GNU的文件，文本，shell工具，基本上这才是LINUX强大的地方。还有各种开源软件，需要什么用什么。

Windows软件推荐

这里有一些在Windows上比较好（很多人可能并不知道它们有那么好）的软件。

Firefox 和 Chrome

最新版本的firefox和chrome功能都非常强大，响应很快，尤其是chrome，真的是飞一般的感觉。其实IE9也不错，只是它只能装在Windows 7 上，这个太不友好了。

真正让firefox和chrome变得强大的是它们的扩展功能，我们可以在上面安装各种插件来实现各种我们想要的功能。举个例子，如果你很讨厌网站上的广告，尤其是视频广告，什么5分钟的视频，有一段30秒或45秒的广告，你肯定疯掉了。没关系，装一个Adblock Plus的插件，一下子世界清净了。

如果没有用过这两个浏览器，那就用一下吧，只有想不到，没有做不到。

幻想影音

一款很棒的本地视频播放器，集成了potplayer，kmplayer，mpc等播放器以及各种解码器，基本上是无所不能。尤其是播放高清视频，有些人喜欢下载720p，1080p的视频，幻想影音播放器起来非常稳定，画质也很好。

其实还有些类似的软件，像完美解码，终极解码，都差不多，但个人感觉还是幻想更好些，不过貌似官方网站不稳定，所以要下载的话，就上百度google一下吧。

Foobar2000

一款超赞的本地音乐播放器，播放音质好，可以随意订制图形界面和播放组件，又非常小巧清爽。千千静听现在走在线路线，播放效果一般，而且有广告，很烦人。唯一可能让你放不下的是它的歌词功能，其实foobar2000很容易做到，装个歌词插件就可以了，推荐autolyric，很不错，一直在用。

Foxit Reader

一款PDF阅读器。不说什么了，万恶的，臃肿的Adobe Reader,好电脑打开它都要卡。还是用Foxit Reader吧，小巧轻便，功能齐全。

Notepad++

非常强大的文本编辑器，如果需要经常和代码打交道，用它就没有错了。

WinSCP

用于远程连接SFTP，SSH，FTP服务器的客户端，配合上上面的Notepad++用来编辑远程服务器的文件特别方便。
